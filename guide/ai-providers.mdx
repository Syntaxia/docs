---
title: "AI providers"
description: "Configure your own AI provider keys for ontology discovery."
---

Syntaxia uses a Bring Your Own Key (BYOK) model for AI. Your organization configures its own LLM API keys from supported providers: Anthropic (Claude), OpenAI (GPT-4o, o3), or xAI (Grok). One provider is active per organization at a time. Usage is tracked per request with token counts and estimated costs.
